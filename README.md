# proj-block
* date : 2019/02/19
* 통합 이모티콘 서비스

## 불편한 점
* 메신저마다의 독자적인 이모티콘 존재
* 구매해서 보유하고 있는 이모티콘을 다른 메신저에서 사용할 수 없음

## 해결 방안
* 이모티콘에 고유 ID를 부여하고 각각의 이모티콘을 블럭 트랜잭션에 넣는다.
* 구매 이력이 있을 경우 다른 메신저에서도 다운받아 사용가능하도록 한다.

### 구성원
* 김세진
